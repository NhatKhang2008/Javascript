# Vòng lặp hữu hạn: for

Cấu trúc vòng lặp for:
for (<Khởi tạo biến lặp>; <Điều kiện>; <Bước nhảy>){
<Logic thỏa mãn điều kiện>
}

- let i = 0: Khởi tạo biến lặp
- i < 10: Điều kiện
- i = i + 1: Bước nhảy
